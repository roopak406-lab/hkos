Place the Aromatic Tadka Kitchen emblem here as:

    logo.png

The app references /brand/logo.png for:
  • the storefront hero, landing page, login, and admin nav (via <BrandLogo>)
  • the PWA install icons (see src/app/manifest.ts)

A square PNG with transparent corners works best (the emblem is circular).
Recommended: at least 512×512 px. The same file is reused for the 192px and
512px icon slots, so one high-res copy is enough.

Until this file is added, the UI shows a green ChefHat fallback badge — nothing
breaks.
